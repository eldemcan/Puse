Welcome to Puse project page

What is Puse?

Puse is an simple sequence diagram drawer which has different approach for generating diagrams.Instead of drawing sequence diagrams in old fashioned way like dragging and dropping items. Puse enables user to give textual commands and generate the correspond graphics immediately.

Which language and frameworks used to develop Puse ?

Puse is written in Java-Eclipse and it uses PlantUml framework for generating sequence diagrams.

Download and Install

You can download Puse from here.

Requirements

  1. Java
  2. Graphviz

Windows

  1. Run setup file

  2. Choose** user file** to install Puse, otherwise Puse will not able to generate diagram.(See image below)

Linux

1.) In order to run the setup file you need to give permissions in your Linux system.

2.) After installation completed. File will be appear on your desktop. It also requires permission to run.

After this command is written, Puse icon on desktop should change.

3.) Right click on shortcut in desktop and choose properties and add java -jar at the beginning of command part.
